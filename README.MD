# Job Portal

A full-stack Job Portal built using:

- React
- Tailwind CSS
- Django REST Framework
- JWT Authentication
- MySQL

## Features

- User Registration
- JWT Login
- Role-Based Authentication
- Candidate Dashboard
- Employer Dashboard
- Create Jobs
- Apply Jobs
- Application Tracking

## Tech Stack

Frontend:
- React
- Tailwind CSS
- Axios

Backend:
- Django
- DRF
- JWT

Database:
- MySQL

## Installation

Backend

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver

Frontend

npm install

npm run dev
